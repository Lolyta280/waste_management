# Waste Management System

## Overview
This Waste Management System is designed to streamline the waste collection process in Penampang, Sabah. The system integrates PHP and MySQL to manage user authentication, collection scheduling, and tracking.

## Features
- User roles: Admin, Collector, and User
- Dashboard tailored to each role
- Schedule collection dates
- Update collection status
- Responsive front-end interface

## Requirements
- PHP 7.0 or higher
- MySQL
- Apache Server (XAMPP/LAMP/WAMP)

## Installation
1. Clone the repository.
2. Import the SQL file to create the database.
3. Configure `db.php` with your database credentials.
4. Start the server and access the application via your web browser.

## Usage
- Visit `login.php` to access the system.
- Admins can create schedules and manage users.
- Collectors can view and update their collection schedules.
- Users can check schedules for their areas.
